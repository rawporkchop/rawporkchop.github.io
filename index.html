About the project

Built for the stage,
not the studio

The LIMIT-1 is a mobile hardware multieffects unit by Arcsin Engineering, designed for guitarists and performers who need professional-grade signal processing without the weight of a traditional pedalboard.

This companion app extends the hardware with a full iOS interface — load impulse responses, arrange AUv3 plugin chains, and map every parameter to the 8 physical knobs on the device. Everything designed for zero-friction live use.

The architecture bridges native SwiftUI with a C++ audio engine via Objective-C++, keeping UI and DSP in their natural environments without compromise.

Core features

Everything you need
at volume

IR Cabinet Loading
Load any .wav impulse response for real-time cabinet simulation. JUCE ConvolutionEngine with background loading — no audio thread blocking.
AUv3 Plugin Chain
Host and reorder AUv3 plugins in a live signal chain. Each plugin runs in its own XPC process — one crash never brings down the host.
Knob Carousel
Full-screen rotary knobs at 88pt minimum — built for gloves, low light, and live stages. Swipe to switch plugins, no tab bar in sight.
Bidirectional MIDI Sync
Physical knob turns update the screen. Screen drags send MIDI CC back to hardware. Feedback loops eliminated with source-tagged parameter updates.
Preset Persistence
SwiftData backs every preset, IR file reference, and plugin chain. Presets survive app restarts, device migrations, and iCloud sync.
Multi-Platform
iOS first, then macOS via Catalyst, then Windows via JUCE. The C++ audio engine is platform-agnostic — only the UI adapts per target.
